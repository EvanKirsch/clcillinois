# Python
Python is a great introductory programming laguage!
It's  easy to read, has forgiving syntax and is multi-paradigm. This makes it the perfect lanaguage for an introduction to programming.
Python is intereperted so there is no need to compile programs before executing them. 

## Running Python files from a bash shell
In order to run a python file we use the `python3` package with the command `python3 <filename>`.

As an example we can run our `Hello_Word.py` file from the with the command `python3 Hello_World.py` from the python directory.
