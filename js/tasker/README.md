## Running a http server in a Github code space bash terminal
1. Install `http-server` using `npm` via the command `npm i -g http-server`
1. Start the http server by running the command `http-server`
1. Click the "open in browser" button on the popup. I _think_ you can navigate to the address by modifying the url of the container to specify the port. ex: `expert-waffle-wg6pjx9jx9hvg7p.github.dev` becomes `expert-waffle-wg6pjx9jx9hvg7p-8080.app.github.dev` via `p` -> `p-8080.app`
1. Navigate to the index file
